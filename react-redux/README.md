# React Redux

A comprehensive guide to advanced react redux.

## Setup (06 - 07 lessons)

Install `json-server` to host the `data/db.json` file and serve its contents as API responses.

```bash
npm i json-server -g
json-server --watch data/db.json --port 3500
```
